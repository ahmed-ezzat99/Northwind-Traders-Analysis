# Northwind Traders Global Sales Analysis
Power BI Project

## Project Objective
The goal of this project is to analyze global sales and customer trends for Northwind Traders using Power BI.  
The focus is on uncovering revenue drivers, evaluating sales performance, and generating actionable business insights through interactive dashboards.

This project demonstrates skills relevant to **Data Analyst** and **BI Analyst** roles, including data modeling, KPI calculation, and dashboard development.

---

## Project Scope
- Analyze total sales by country and year
- Track sales performance by product and category
- Monitor orders and revenue contribution by employees
- Assess shipping methods and their impact on sales
- Present insights using interactive Power BI visualizations

---

## Tools & Technologies
- Power BI Desktop
- DAX for measures and calculations
- Data modeling and relationships
- Interactive dashboard design

---

## Project Deliverables
- Cleaned and modeled dataset in Power BI
- Interactive Power BI dashboard with global sales insights
- Business-focused insights and recommendations
