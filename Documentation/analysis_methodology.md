# Analysis Methodology

## 1. Data Modeling
- Imported all relevant tables (Customers, Orders, Products, Employees, Shippers, Dates)
- Defined primary and foreign key relationships
- Created relationships between tables in Power BI

## 2. Measures & KPIs
- Calculated Total Sales, Quantity, and Order Count using DAX
- Created KPIs for sales by country, year, product, and employee

## 3. Dashboard Design
- Built interactive visuals for country-wise and year-wise sales
- Displayed top companies and employees by order count
- Included filters for Country and Year
- Highlighted shipping impact and regional performance

## 4. Insights Generation
- Identified top-performing countries, employees, and companies
- Monitored global sales trends
- Assessed influence of shipping methods on order volume
- Provided actionable recommendations based on insights
